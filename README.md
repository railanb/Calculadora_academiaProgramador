# Calculadora

Primeiro projeto feito durante curso da [ACADEMIA DO PROGRAMADOR](https://www.academiadoprogramador.net/inicio) 2026

![Giff do programa](/giff/apresentacao.gif)

## Introdução
Esse projeto foi desenvolvido usando a linguagem C#, uma aplicação de consoles simples, sua funcionalidade tem como calcular 2 valores numericos inteiros, permitindo realizar as 4 operações matemáticas. Também tem a visualização de histporico das operações e visualização da tabuada.

## Funcionalidades
- **Operações Básicas**: Realize somas, subtrações, multiplicações e divisões.
- **Tabuada**: A calculadora é capaz de gerar a tabuada  de 1 à 10 de um número informado pelo usuário, também é possível selecionar a operação matemática.
- **Historico de Operações**: A calculadora é capaz de armazenar na memória um histórico das operações anteriores.

## Como utilizar o programa
1. Clone o repositório ou baixe os arquivos do repositório.
2. Abra seu emulador  de terminal de preferência e navegue até a pasta raiz do projeto baixado.
3. Utilize o comando abaixo para restaurar as dependências  do projeto.
```
dotnet restore
```
4. Em seguida compile e execute o projeto com o comando:
```
dotnet run --project Calculadora.ConsoleApp
```

## Requisitos
- [.NET 10.0 SDK](https://dotnet.microsoft.com/pt-br/download)
